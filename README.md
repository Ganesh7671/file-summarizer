# 📄 File Summarizer – AI Powered Document Summarization System

An AI-powered Spring Boot web application that allows users to upload documents (PDF, images) and automatically generate concise summaries using Large Language Model (LLM) integration.

---

## 🚀 Features

- 📂 File Upload Support (PDF / Image)
- 📝 Text Extraction from Documents
- 🤖 AI-Based Intelligent Summarization
- ⚠ Global Exception Handling
- 🧱 Clean Layered Architecture (Controller → Service → Repository)
- 🌐 REST API Based Backend

---

## 🛠 Tech Stack

- **Java 17**
- **Spring Boot**
- **Maven**
- **REST APIs**
- **Spring WebClient**
- **Ollama / LLM Integration**
- **HTML (Frontend UI)**

---

## 🏗 Architecture

Controller Layer  
Service Layer  
Repository Layer  
Configuration Layer  
Exception Handling Layer
